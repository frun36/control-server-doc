inlineprotected

Sends IPbusControlPacket p and waits for a response.

  • Checks if IPbusTarget is online - false if not
  • Checks if request is empty - true if yes, does nothing
  • Writes to qsocket - false if error
  • Reads from qsocket - false if error or response incorrect
Parameters
[in,out]pThe control packet being sent. After response is received it is also stored within p
shouldResponseBeProcessedindicates whether processResponse should be called to verify the response
Returns
true if the operation was successful, false otherwise
