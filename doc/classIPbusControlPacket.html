Detailed Description

A class responsible for creating a single IPbus packet and checking the corectness of the response.

IPbusControlPacket stacks one or more IPbus transactions into a single packet. See the IPbus specification section 3

Adds transaction to the packet.

addTransaction saves transaction in two formats: within transactionsList as Transaction object and within the request buffer in a format appropriate for IPbus communication. Transactions within a packet are stored in the following manner (values in [] represent bit numbers):

  • [0-31] - packet header
  • [32-63] - transaction header
  • [64-95] - destination address
  • [96-...] - data
  • [x-x+32] - next transaction header
  • ... Number of transactions within one packet is limited by the maximum packet size.
Parameters
typeType of transaction
addressAddress of a memory location at the remote site on which operation will be performed
dataAddress of a memory block of data passsed to the transaction
nWordsSize of a data block specified in words

A method that creates a mask used in the RMWbit transaction.

RMWbit operation is performed according to the equation: X = (X & A)\B, where X is a word (register) on which operation is performed. Mask is saved in the dt field, AND term is saved as the first element, OR term is saved as the second.

Parameters
mask0- AND term of a RMWbit transaction
mask1- OR term of a RMWbit transaction
Returns
Pointer to the mask (dt field)
