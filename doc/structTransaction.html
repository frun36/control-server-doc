Detailed Description

A struct containing full information about a single transaction.

is represented within the IPbus packet by three components: transaction header (1 word), Address of the memory location on which the operation will be performed (1 word) and a block of data (if any data is required). Data layout is speficic for each kind of transaction. Transaction stores information about request, also after the response is received pointer to the response header will be stored in field responseHeader.
